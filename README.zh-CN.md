# <img src="Editor/Logo%40Sphere%20(Custom).png" alt="LAuX" width="32" align="top">SimpleMix

**SimpleMix** 是一个轻量的 Unity / VRChat 世界音频混音工具，用于制作分层环境音、区域过渡和简单 SFX 触发。

[English README](README.md)

## 快速开始

使用 `LAuX > Simple Mix > Setup Scene` 创建运行和编辑结构。将循环音源放到 `Layers` 下，将 SFX 音源放到 `SFX` 下，然后在 SimpleMix 的 Inspector 中编辑 snapshots 和 triggers。

可选 Debug UI 可以通过 `LAuX > Simple Mix > Place Debug Panel` 放置。
